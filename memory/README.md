# memory/

Durable project memory for coding agents.

| File | Purpose |
|---|---|
| `project.md` | Stable facts about the project: purpose, stack, conventions. |
| `handoff.md` | Current working state for the next session or agent. |
| `reflections/` | Dated notes, decisions, and lessons learned. |

## Rules
- Keep entries concise and factual.
- Never store secrets, tokens, or credentials here.
- Update `handoff.md` at the end of a significant work session.
