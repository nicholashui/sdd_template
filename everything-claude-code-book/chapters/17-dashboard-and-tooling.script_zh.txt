第十七章。仪表板和生态系统工具。

ECC 大部分是一个终端体验，但它也内置一道图形前门，和一个小小的配套工具生态系统。这一个短章讲那些友善的界面。

我们从桌面仪表板开始讲。有一个桌面图形界面，用 Tkinter 工具包构建，让你可视化地探索目录里面所有东西。你通过一个 npm 脚本，或者直接执行那个 Python 文件来启动它。它的功能包括一个给 agents、skills、commands、rules 和设置用的标签界面；横跨所有组件的搜索和过滤；一个深色和浅色主题切换；字体家族和大小的自定义；和在头部和任务栏的项目标志。它是浏览 64 个 agents 和 262 个 skills 的最容易方法，不用打印出 Markdown 文件，也是一个新队友定位自己的好方法。它需要 Python 3，并且有 Tkinter 工具包可用。如果那个 npm 脚本找不到它，你就直接执行那个 Python 文件，并且确认你的 Python 装了 Tkinter。

接着，技能创建器。有两种方式从一个 repo 生成 skills，我们之前都碰过。本机、内置那种，在本机分析你的 git 历史并且写 skill 文件，有一个顺便为第二代学习系统播种的选项，而且没有外部服务。进阶那种是一个 GitHub App，给超大型 repo、自动拉取请求，和团队分享，通过一个托管的 marketplace 应用。两者都从你的 commit 历史，产出即用的 skills、instinct 集合，和模式抽取。

有 AgentShield，我们在安全那一章深入讲过。一句总结：它是一个给你的代理配置用的安全审计器，用一个单一命令就能运行，有一个深度红队、蓝队和审计员模式，和一个对持续集成友善的退出码。在框架里面，你把它当成一个安全扫描来运行。

还有一群 skill 和框架健康工具，随时间让你的设置保持诚实。有一个带分析的 skill-health 仪表板。有一个 skill-stocktake 去审计 skills 和 commands 的质量。有一个 harness-audit 去为框架可靠性、评估就绪度和风险评分。还有一个 rules-distill command 去从 skills 抽取横跨各处的原则，并且把它们变成 rules。有命令行版本的 harness audit、一个可观测性就绪度检查，和一个操作者就绪度仪表板。随着你的目录成长，这些就是你防止 skill 蔓延和配置漂移的方法。


至于包管理器的设置：ECC 自动检测你的包管理器，无论是 npm、pnpm、yarn，还是 bun，按一个优先次序，从一个环境变量开始，然后一个项目配置文件，然后你项目文件里面的 package-manager 字段，然后一个锁定文件，然后全局配置，最后就是第一个可用的。你也可以明确地设置它，用一个环境变量、一个全局或者项目的设置 command、一个 detect command，或者一个框架内的设置 command。

有一个关于编辑器配搭的生产力提示，因为编辑器塑造体验。短篇指南在这里有它的立场。它的首选是 Zed，它是 Rust 那么快、通过它的代理面板实时追踪文件改动、有一个快速命令面板，并且用低资源，这一点在运行最强大的模型的时候很重要。VS Code 和 Cursor 都运作得好，用终端配 IDE 集成来做语言服务器，或者用集成的扩展。不分编辑器的贴士：把你的屏幕在助手和编辑器之间分割、启用自动保存，这样那些文件读取才是最新的、用 git 集成在 commit 之前审查差异，并且确认文件监视器会自动重新加载改动了的文件。

这里是生态系统一览。桌面仪表板是一个浏览目录的图形界面。技能创建器从 git 历史生成 skills。AgentShield 是安全审计器。托管的 GitHub App 提供托管分析、拉取请求审计，和私有 repo 支持。持久的 REPL 给你路由和度量。操作者命令行界面处理操作者和生命周期的工作。而 Rust 控制平面是 ECC 2.0 的 alpha。开源 repo 是 MIT 许可证，永远免费。付费的部分，也就是托管的 GitHub App 和赞助，资助这份工作，这就是为什么一个单一的维护者，可以每个星期跨很多框架出货。要用本书里面所有东西，你永远不需要付费的部分。

那么，重点精华。仪表板脚本打开一个浏览 agents、skills、commands 和 rules 的图形界面。Skill-create command 在本机从你的 git 历史生成 skills，不用任何服务。你用 skill-health、skill-stocktake、harness-audit 和 rules-distill 工具，保持你的设置健康。你通过设置 command 或者一个环境变量去设置你的包管理器。而开源核心是免费的，至于托管的工具就是可选的。
